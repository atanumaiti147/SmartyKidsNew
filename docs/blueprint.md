# **App Name**: SmartyKids - Fun Learning for Nursery

## Core Features:

- Interactive Home Screen: A welcoming screen with an animated mascot, clear buttons for different sections, and background music controls.
- Graded Lessons & Levels: Curated topics like alphabets, numbers, colors, animals, and shapes, each with multiple levels that increase in difficulty. Includes a level unlock system.
- Varied Question Types: Offers multiple-choice and fill-in-the-blank questions with image aids and audio narration. Includes a tip tool after incorrect answers, offering friendly guidance.
- Rewards & Motivation System: An engaging reward system that offers stars, badges, coins, and gems for completing levels, plus a reward shop for stickers and background themes.
- Timed Challenges: Countdown timer that shows encouraging messages as levels progress
- C/C++ Powered Mini-Games: Integrate mini-games built with C/C++ and compiled to WebAssembly for interactive and performance-enhanced learning experiences.
- Admin Question Editor: Simple JSON admin panel to allow administrators to edit questions.  Panel is password protected.

## Style Guidelines:

- Primary color: Soft, inviting pink (#FFD7D7) to evoke a sense of warmth and playfulness.
- Background color: Pale yellow (#FFF4C3) that's light and cheerful without being distracting.
- Accent color: Light green (#C2F0C2) used for positive feedback elements, such as correct answer indicators and reward highlights, to add a touch of freshness and positivity.
- Font: 'Baloo', a rounded child-friendly font, for headlines and body text; Note: currently only Google Fonts are supported.
- Use animated SVGs and emojis throughout the app to enhance visual appeal and engagement.
- Design with rounded corners (20px border-radius) and subtle drop shadows for buttons to create a soft, inviting interface.
- Include confetti animations for correct answers and animated progress bars to visually represent learning progress.
